Project Highlights

  • Modern, Responsive E-Commerce Platform: Developed a fully responsive online clothing store interface, ShopSphere, ensuring cross-browser compatibility.
  • Dynamic Product Management System: Admin panel allows adding, editing, deleting, searching, filtering, and sorting products with image upload and live preview. Works seamlessly in both standalone product page and admin dashboard.
  • Role-Based Access Control: Implemented admin and user roles with secure login and session handling. Only admins can access dashboard features like product, user, order, and inventory management.
  • Dynamic Dashboard Loading: Main content loads dynamically in the admin dashboard to prevent full page reloads. Sidebar navigation works with product management, profile, and other modules.
  • User-Friendly Features: Includes scroll-to-top button, theme toggle (light/dark mode) and real-time notifications for actions.
  • Secure and Modular Codebase: Passwords hashed with SHA-256, utility functions modularized for reusability, and dynamic JS loading ensures modular management of pages.
  • Team Collaboration: Clear division of tasks among team members for home page, about page, dashboard, checkout, and authentication modules. Structured HTML, CSS, and JS ensure maintainable workflow.

Challenges Faced

  • Dynamic Content Loading:While loading product management and profile pages dynamically in the dashboard it becomes very challenging to update the DOM without breaking functionality or losing JS events.
  • Cross-Page Data Consistency: Maintaining product list and user session data across multiple pages and admin/user views while avoiding duplication or data loss.
  • Responsive Design: Making the website fully responsive across different devices and screen sizes was really challenging.
  • Dark Mode Implementation: Ensuring dark mode applies consistently across body and header home and product page was little bit hard.
  • Image Handling: Uploading and previewing images locally, handling base64 encoding, and displaying them correctly in both admin and product pages.
  • Role-Based Access Control: Securing the admin dashboard and certain features to prevent unauthorized access while managing login sessions effectively was ver hard.
  • Team Collaboration: Coordinating between group members developers on different modules like home, checkout, dashboard, and authentication without code conflicts was really challenging.
  • Frontend Validation and Security: Validating user input (name, email, password) and hashing passwords using SHA-256 to prevent security vulnerabilities was technical.
